# 🚀 Project Manager (نظام إدارة المشاريع)

نظام ويب متكامل واحترافي لإدارة المشاريع التقنية، العملاء، النطاقات، وفريق العمل. تم بناء هذا النظام كلوحة تحكم (Dashboard) متطورة لتسهيل متابعة إنجاز المهام وحالة المواقع الإلكترونية للعملاء بكفاءة عالية.

## ✨ المميزات الرئيسية (Features)
* **لوحة قيادة تفاعلية (Dashboard):** عرض إحصائيات حية لعدد العملاء والمشاريع الحالية.
* **إدارة العملاء (Clients Management):** تسجيل بيانات العملاء الأساسية (CRUD Operations).
* **إدارة المواقع والنطاقات (Websites & Domains):** تسجيل تفاصيل المواقع، ربط كل موقع بعميل محدد تلقائياً، وتتبع حالة الاستضافة والمواعيد النهائية.
* **إدارة فريق العمل والمهام (Team & Tasks):** إضافة أعضاء الفريق وتوزيع المهام البرمجية عليهم.
* **تصميم عصري ومتجاوب:** واجهة مستخدم (UI) احترافية مبنية باللغة العربية، متجاوبة مع جميع الشاشات.

## 🛠️ التقنيات المستخدمة (Tech Stack)
* **الواجهة الخلفية (Backend):** C#, ASP.NET Core MVC
* **قواعد البيانات (Database):** SQL Server, Entity Framework Core (Code-First Approach)
* **الواجهة الأمامية (Frontend):** HTML5, CSS3, Bootstrap 5, Razor Views
